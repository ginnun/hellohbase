This is a sample WebUI for a sample twitter clone. Wicket, Guice and HBase are used. Can be easily converted to one of JSF, Spring, Restful WS. In HBase, I tried to implement Tall-Narrow tables instead of Flat-Wide tables. There are some unnecessary hits to HBase, but those can be eliminated by holding user id in hidden fields at html froms.
